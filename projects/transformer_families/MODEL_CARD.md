# Transformer Families Model Card

The project contains three educational models sharing width, attention-head count, and
block implementation. They are trained only on deterministic synthetic diagnostics.

Intended use: learn attention visibility, objectives, tensor flow, and architectural
selection. Out of scope: language quality, production inference, factual answering,
safety claims, or benchmark comparisons with pretrained models.

The generated report records configuration, seed, parameter counts, initial/final
losses, task accuracies, cross-attention shape, and limitations.
